Identity

Brand & Logo

Name

The product is called Lens. One word, capital L, no suffix. Not LENS, not Lenses, not Lens SF. The wordmark uses Inter 800 at 18px with letter-spacing: -0.04em.

Logo mark

A 2×2 grid of four rounded squares (34×34px total, each square 15×15px, 4px gap, rx 2). The colors are not decorative — they track the active lens.

Writing

Voice & Copy

Precise and direct. Not bureaucratic, not casual. An analyst should be able to quote any label in a public hearing without it sounding off.

Banned phrases

Don't use "officer-initiated" in UI copy. Use "police-initiated" or "police stops."
Don't use "victim-reported" in UI copy. Use "crimes reported by residents."
Don't use "proactive" in UI copy.
Don't use "enforcement ratio" in UI copy. Use "Police Stops per Crime Report."
Don't use "Lens 1", "Lens 2", "Lens 3" in UI copy. These are internal names only.

Flag copy rule

Flags say what is anomalous and name the reference frame. Never a verdict on the neighborhood.

Good: "Some incidents couldn't be placed on the map — counts here may understate actual activity."
"This neighborhood has poor data quality." — that's a judgment, not a description.
